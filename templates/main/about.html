The PasswordGenerator App

The PasswordGenerator app helps to generate random passowords. the application is powered by Django and python.

however, application contains server side interactions, so please do not use this passwords for long, at sensitive

places like online banking etc. note- the PasswordGenerator app backend code

is developed by the developer and do not uses any third party libraries(uses few python in-builts).
